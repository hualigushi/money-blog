**垂直方向的padding**: 在css中，padding-top或padding-bottom的百分比值是根据容器的width来计算的。

```
.wrap{
       position: relative;
       height: 0;  //容器的height设置为0
       width: 100%;
       padding-top: 75%;  //100%*3/4
}
.wrap > *{
       position: absolute;//容器的内容的所有元素absolute,然子元素内容都将被padding挤出容器
       left: 0;
       top: 0;
       width: 100%;
       height: 100%;
}

+ **padding & calc()**: 跟第一种方法原理相同
