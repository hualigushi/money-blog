# TypeScript 类型系统简述
## TypeScript 的类型是支持定义 "函数定义" 的
TypeScript 中类型系统中的的函数被称作 泛型操作符，其定义的简单的方式就是使用 type 关键字：
```js
// 这里我们就定义了一个最简单的泛型操作符
type foo<T> = T;
```

```js
// 把上面的类型代码转换成 `JavaScript` 代码
function foo(T) {
    return T
}
```
